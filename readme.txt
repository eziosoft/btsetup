This project Arduino sketch allows to easily set up Bluetooth module for MultiWii.

Setup:
1.Enter desired Bluetooth name and pin code
2.Upload the btsetup.ino sketch (https://raw.github.com/eziosoft/btsetup/master/btsetup.ino) to your MultiWii board
3.with Bluetooth connected and programing cable disconected repower the board.
4.wait for LED (about 20sek)

It has been tested on AtmelMega328. May not work on Megaboards and Leonardo.
